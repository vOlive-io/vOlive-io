My game dev process

My game dev process is divided into 5 phases based on how complete they are and how enjoyable they are compared to their final product.

Phase 1:

The game idea is made, and the folder is made. Game ideas are drafted, and coding begins.

Phase 2:

Basic variables are made, and the most basic functions start to get made. The is seemingly blank, and is mostly a console game.

Phase 3:

HTML, CSS, annd JS start coming together to form a game, not very fun, but it works.

Phase 4:

The game is now playable and work, with little to no error. This is the skelliton and the full game starts to take shape.

Phase 5:

The game is now release ready and fully playable, the game will continue to get updates.
